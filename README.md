# Dragonfly (dragonfly)
A modern in-memory datastore built for performance and scalability, designed as a drop-in replacement for Redis and Memcached with improved efficiency and throughput. Dragonfly uses a multi-threaded, shared-nothing architecture to deliver significantly higher throughput with lower latency than Redis.
